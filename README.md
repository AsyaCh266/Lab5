# Scrum Project Lab

Этот репозиторий используется для моделирования Scrum-процесса в рамках лабораторной работы.
Проект представляет собой простое приложение для управления заметками.
Цель — отработать работу с GitHub Issues.
Все требования описываются в виде задач (issues).
Работа ведётся в рамках одного спринта.

# Sprint Review

В рамках спринта были закрыты задачи по созданию структуры проекта и базовой функциональности.
Была реализована основная логика приложения.
Часть задач, связанных с дополнительными возможностями, не была завершена.

# Sprint Retrospective

Что прошло хорошо:
Чёткое планирование задач и понятные критерии готовности.

Что пошло не так:
Некоторые задачи были недооценены по сложности.

Что можно улучшить:
Более точно оценивать story points и дробить крупные задачи.
